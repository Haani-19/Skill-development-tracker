# 📌 Skill Development Tracker  
### Web Technology Mini Project

## 📖 Overview  
The **Skill Development Tracker** is a web-based application developed as part of a **Web Technology mini project**. It helps users organize, manage, and track their learning activities in a structured way.

The system allows users to add tasks related to different skills, set deadlines, and monitor their progress. It also provides a visual representation of task completion, making it easier to stay consistent and productive.

---

## 🎯 Objectives  
- To develop a simple web application using core web technologies  
- To help users track skill-based tasks efficiently  
- To implement CRUD operations using PHP and MySQL  
- To demonstrate integration of XML with web applications  
- To provide a visual progress tracking system  

---

## 🚀 Features  
- Add, update, and delete tasks  
- Skill-based task organization (using XML)  
- Deadline management  
- Task status tracking (Pending / Completed)  
- Separate pages for better navigation  
- Progress visualization using graphical bars  
- Simple and user-friendly interface  

---

## 🛠️ Technologies Used  

- **HTML** – Structure of web pages  
- **CSS** – Styling and layout design  
- **PHP** – Backend logic and processing  
- **MySQL** – Database for storing tasks  
- **XML** – Storage of predefined skill categories  
- **XAMPP** – Local server environment  

---

## 🧱 Project Structure  


skill_tracker/

---

## ⚙️ How to Run the Project  

1. Install **XAMPP**  
2. Start **Apache** and **MySQL**  
3. Copy the project folder into:  

C:\xampp\htdocs\

4. Open browser and go to:  

http://localhost/skill_tracker/

5. Create database `skill_tracker` in phpMyAdmin  
6. Import or create the required `tasks` table  
7. Run the application  

---

## 📊 Working  

- User adds a task with skill and deadline  
- PHP processes the input and stores it in MySQL  
- Tasks are displayed dynamically on the page  
- User can update or delete tasks  
- Progress page shows completion percentage using a graph
